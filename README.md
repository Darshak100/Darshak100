## 🔥 Featured Project: Handwritten Digit Generation (DCGAN)

> Achieved **FID = 5.94** and **87.83% downstream accuracy** using a DCGAN trained on MNIST.

| Metric | Value |
|--------|-------|
| Fréchet Inception Distance (FID) | **5.94** (Excellent) |
| Inception Score (IS) | 2.44 ± 0.036 |
| CNN trained on generated data → real test accuracy | **87.83%** |
| Training epochs | 100 |

### 🎯 Key Highlights
- 🧠 **Architecture:** Deep Convolutional GAN with label smoothing, batch norm, and dropout
- 📊 **Evaluation:** FID, Inception Score, t‑SNE, latent interpolation, and pixel histograms
- 💡 **Impact:** Synthetic digits can replace real data for OCR – practical data augmentation
